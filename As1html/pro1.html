b tag

The b tag in HTML is used to draw reader's attention quickly and it shows that the content is important and serious. This shows the content in boldface.


i tag

The i tag in HTML is used to display the text in itallic style or as idiomatic text like This is i tag


u tag

The u tag is used to render the text with underline like this is u tag


select tag

The select tag is used to represent control which provides menu of options.


button tag

The button tag is used to provide a button for a user where it can activated by the user


hr tag

The hr tag is user to provide a horizontal line such as after the paragraph or after a section and many more.


br tag

The br tag is used to provide a single line break wherever it is needed.
